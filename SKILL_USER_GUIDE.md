# AI Skill功能使用指南

## 快速开始

### 如何打开Skill管理器
1. 登录系统
2. 进入AI助手页面
3. 点击顶部栏的📖书本图标（位于历史对话按钮左侧）

### 为什么我看到的技能数量和别人不一样？

**这是正常的！** 系统根据您的角色显示相应的技能。

| 角色 | 技能数量 | 说明 |
|------|---------|------|
| 用户 (Client) | 12个 | 营养、健康、报告等 |
| 家属 (Guardian) | 12个 | 照护、监测、协调等 |
| 政府 (Gov) | 11个 | 政策、数据、监管等 |
| 商家 (Merchant) | 2个 | 菜品、采购等 |

## 技能类型

### 🌟 系统技能
- 带星标图标
- 灰色渐变背景
- 标签显示"系统"
- **只能查看，不能编辑或删除**

### ✨ 自定义技能
- 绿色标签显示"自定义"
- 可以查看、编辑、删除
- 文件保存在后端Skills文件夹

## 功能说明

### 查看技能
1. 在技能列表中找到想查看的技能
2. 点击"查看"按钮
3. 弹出对话框显示技能的完整内容

### 创建自定义技能
1. 点击右上角"创建技能"按钮
2. 输入技能名称
3. 编写技能内容（支持Markdown格式）
4. 点击"创建"按钮

**技能内容模板：**
```markdown
# 技能名称

## 角色定位
描述AI的角色和职责

## 任务要求
具体的任务描述和输入数据格式

## 输出内容
期望的输出内容和格式

## 语气要求
对AI回复语气的要求
```

### 编辑自定义技能
1. 找到您创建的自定义技能
2. 点击"编辑"按钮
3. 修改技能内容
4. 点击"保存"按钮

### 删除自定义技能
1. 找到您创建的自定义技能
2. 点击"删除"按钮
3. 确认删除操作

## 变量使用

技能内容中可以使用变量，系统会自动替换为实际数据：

### 营养数据变量
- `{calories}` - 热量
- `{protein}` - 蛋白质
- `{fat}` - 脂肪
- `{carbs}` - 碳水化合物
- `{fiber}` - 膳食纤维
- `{calcium}` - 钙
- `{vitaminC}` - 维生素C
- `{iron}` - 铁

### 目标数据变量
- `{dailyTarget}` - 每日目标
- `{achievement}` - 达成率
- `{newDishes}` - 新菜品数量

### 用户信息变量
- `{clientName}` - 用户姓名
- `{age}` - 年龄
- `{gender}` - 性别
- `{chronicConditions}` - 慢性病情况

### 其他变量
- `{solarTerm}` - 节气
- `{period}` - 时间段

**使用示例：**
```markdown
## 分析要求
请根据以下数据进行分析：
- 总热量：{calories} kcal
- 蛋白质：{protein} g
- 每日目标：{dailyTarget} kcal
- 达成率：{achievement}%
```

## 常见问题

### Q: 为什么我只看到几个技能？
A: 这是正常的！系统根据您的角色显示相应的技能。不同角色看到的技能数量不同。

### Q: 为什么我不能编辑某些技能？
A: 带星标⭐的系统技能是预设的，只能查看不能编辑。您可以创建自己的自定义技能。

### Q: 自定义技能保存在哪里？
A: 自定义技能保存在后端的 `Backed/Skills/` 文件夹中，文件名格式为 `custom-{角色}-{名称}.md`

### Q: 我创建的技能别人能看到吗？
A: 不能。每个用户只能看到和管理自己角色的技能。

### Q: 如何让AI使用我的自定义技能？
A: 创建技能后，AI会自动识别并使用。您可以在对话中提到技能相关的内容，AI会根据技能提示词回复。

### Q: 技能内容支持什么格式？
A: 支持Markdown格式，包括标题、列表、粗体、斜体等。

## 技能示例

### 示例1：营养分析技能
```markdown
# 营养分析师

## 角色定位
你是一位专业的营养师，擅长分析饮食数据并提供科学建议。

## 任务要求
根据用户的饮食数据：
- 总热量：{calories} kcal
- 蛋白质：{protein} g
- 脂肪：{fat} g
- 碳水化合物：{carbs} g

## 输出内容
1. 营养摄入评估
2. 改善建议（3-5条）
3. 推荐食物类型

## 语气要求
温和、鼓励、专业
```

### 示例2：健康报告技能
```markdown
# 健康周报生成

## 角色定位
你是健康数据分析专家，负责生成周报。

## 任务要求
分析本周数据：
- 时间范围：本周
- 目标达成率：{achievement}%
- 新尝试菜品：{newDishes} 种

## 输出内容
1. 本周总结
2. 亮点与进步
3. 需要改进的地方
4. 下周建议

## 语气要求
积极、鼓励、具体
```

## 技术支持

2. 确认已登录且角色正确
3. 确认在AI助手页面
4. 刷新页面重试
