【问题回顾】
----------
✗ 之前的程序生成的点云全是白色/空白
✗ 原因: SDK的PointCloudFilter返回的数据全是(0,0,0)
✗ 尝试了多种方法都无法解决

【根本原因】
----------
Python SDK的PointCloudFilter有问题，无法正确生成点云数据。

但C++ SDK有正确的实现方法！

【解决方案】
----------
✓ 参考C++ SDK的GeneratePointCloud.cpp
✓ 实现了正确的深度到点云转换
✓ 使用相机内参手动计算3D坐标

【立即使用】
----------

步骤1: 测试C++工具（可选，验证相机正常）
---------------------------------------
运行: 测试新SDK.bat
选择 [1] SimpleViewer 查看深度图
或
选择 [2] GeneratePointCloud 生成点云文件

步骤2: 使用Python程序（推荐）
----------------------------
运行: 启动_正确版本.bat

等待深度窗口打开
按 R 键生成点云
查看3D点云窗口

【关键改进】
----------

1. 获取相机内参
   - 从Pipeline.getCameraParam()获取
   - fx, fy, cx, cy

2. 手动转换深度图
   - 遍历每个像素
   - 过滤无效深度 (20-4000mm)
   - 计算3D坐标

3. 正确的坐标公式
   world_x = depth * (u - cx) / fx
   world_y = depth * (v - cy) / fy
   world_z = depth
