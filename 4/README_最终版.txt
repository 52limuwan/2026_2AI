第1步: 安装 Open3D
  conda activate orbbec_env
  pip install open3d

第2步: 运行程序

  会自动弹出 2 个窗口:
    • Depth - 深度图
    • Point Cloud - 3D点云

Q: 点云窗口没有显示?
A: 需要安装 Open3D: pip install open3d

Q: 点云显示是黑色的?
A: 正常，等待几秒相机预热

Q: 点云更新很慢?
A: 正常现象，实时处理需要计算

Q: 怎么保存点云?
A: 使用另一个程序 orbbec_viewer.py
