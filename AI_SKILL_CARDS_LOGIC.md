# 技能卡片显示逻辑

## 显示规则

### 场景 1: 检测到技能关键词
显示完整的技能卡片流程（3个卡片）：

```
用户问题: "我有高血压，饮食上需要注意什么？"
    ↓
┌─ 技能卡片 1 ─────────┐
│ 思考中 → 思考已完成   │
└──────────────────────┘
┌─ 技能卡片 2 ─────────┐
│ 阅读 SKILL → 读取技能 │
│ ls /app/.sshb/skills/ │
│ → 慢病管理师.md       │
└──────────────────────┘
┌─ 技能卡片 3 ─────────┐
│ 使用技能              │
│ 慢病管理师 - 慢性病管理│
└──────────────────────┘
AI 回复: "高血压患者应该..."
```

### 场景 2: 未检测到技能关键词
只显示思考卡片（1个卡片）：

```
用户问题: "你是谁"
    ↓
┌─ 技能卡片 1 ─────────┐
│ 思考中 → 思考已完成   │
└──────────────────────┘
AI 回复: "我是您的 AI 营养顾问..."
```

## 实现逻辑

### 1. 技能检测

```javascript
const detectedSkill = detectSkill(content)

if (detectedSkill) {
  // 场景 1: 检测到技能
  messageSkillSteps = [
    { type: 'thinking', title: '思考中', skillName: '' },
    { type: 'read', title: '阅读 SKILL', skillName: '', isTransitioning: false },
    { type: 'use', title: '使用技能', skillName: detectedSkill }
  ]
  
  // 启动完整的动画流程
  startSkillAnimationForMessage(userMessageIndex)
} else {
  // 场景 2: 未检测到技能
  messageSkillSteps = [
    { type: 'thinking', title: '思考中', skillName: '' }
  ]
  
  // 不启动动画，只显示思考卡片
}
```

### 2. 状态更新

```javascript
// AI 回复后，更新思考状态
if (messages.value[userMessageIndex].skillSteps) {
  const thinkingStep = messages.value[userMessageIndex].skillSteps.find(s => s.type === 'thinking')
  if (thinkingStep) {
    thinkingStep.title = '思考已完成'
    thinkingStep.type = 'thinking-done'
  }
}
```

8. **季节养护师 - 四季养生**
   - 关键词：春季、夏季、秋季、冬季、节气...

9. **居家护理师 - 日常护理指导**
   - 关键词：护理、照顾、照料、料理、日常护理...

    - 关键词：体检、检查、化验、报告、体检报告...

11. **数据分析师 - 用户月报分析**
    - 关键词：月报、月度、本月、上月...

12. **数据分析师 - 用户周报分析**
    - 关键词：周报、本周、上周、这周...

### 匹配规则
- 基于关键词匹配，计算每个技能的匹配分数
- 选择匹配分数最高的技能
- 如果没有任何关键词匹配，则不使用技能

## 优势

### 1. 清晰的视觉反馈
- 有技能：用户知道 AI 使用了专业技能
- 无技能：用户知道 AI 在通用对话模式

### 2. 减少视觉噪音
- 不是所有消息都需要完整的技能卡片
- 简单问题只显示思考状态

### 3. 更好的用户体验
- 技能卡片有意义时才显示
- 避免不必要的信息干扰

### 4. 灵活性
- 可以根据需要调整显示逻辑
- 未来可以添加更多卡片类型

## 注意事项

### 1. 技能检测准确性
- 依赖关键词匹配，可能有误判
- 建议定期优化关键词列表

### 2. 动画性能
- 只有检测到技能才启动动画
- 减少不必要的动画开销

### 3. 数据一致性
- 技能卡片信息保存到数据库
- 刷新页面后状态保持一致

### 4. 历史消息
- 历史消息直接显示最终状态
- 不重新播放动画
