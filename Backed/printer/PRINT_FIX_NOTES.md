# 打印问题修复说明

## 问题描述
在连续打印 a1.png 和 a2.png 时，第一张图片有概率只打印一半，在正方形标签纸上出现两个半截图片，但第二张打印正常。

## 根本原因
1. **标签定位问题**：打印机在开始打印前，标签纸可能不在正确的起始位置
2. **缓冲区同步问题**：连续打印时，第二个打印任务可能在第一个还未完成时就开始
3. **打印机状态未就绪**：首次打印时打印机可能还在初始化状态

## 解决方案

### 1. 添加 HOME 命令（标签定位）
在 `label_printer_dll.py` 的 `print_label()` 方法中，在设置标签尺寸后添加：
```python
self.tsc.sendcommand('HOME')
```
这会让打印机走纸到下一个标签的起始位置，确保打印从正确位置开始。

### 2. 添加打印间隔延迟
在 `print_order.py` 中：
- 打印前添加初始定位和延迟（0.3秒）
- 两次打印之间添加延迟（1.5秒）

```python
# 打印前定位
printer.tsc.sendcommand('HOME')
time.sleep(0.3)

# 打印第一张
printer.print_label(label1)

# 等待完成
time.sleep(1.5)

# 打印第二张
printer.print_label(label2)
```

### 3. 延迟时间说明
- **0.3秒**：HOME 命令执行时间，让打印机完成标签定位
- **1.5秒**：打印完成时间，包括图片处理、打印、走纸的总时间

- 第二张标签完整打印 a2.png
- 不再出现半截图片重复的情况
- 每张标签都从正确位置开始打印

## 如果问题仍然存在

### 调整延迟时间
如果打印机速度较慢，可以增加延迟：
```python
time.sleep(2.0)  # 从 1.5 增加到 2.0 秒
```

### 检查标签设置
确认标签尺寸和间隙设置正确：
```python
LabelData(width=50, height=50, gap=2, density=8, copies=1)
```
- width/height: 标签尺寸（毫米）
- gap: 标签间隙（毫米）
- density: 打印密度（0-15，推荐 8）

### 打印机校准
如果问题持续，可能需要校准打印机：
1. 关闭打印机
2. 按住 FEED 键开机
3. 等待打印机自动校准标签位置
4. 校准完成后松开按键

## 技术细节

### HOME 命令作用
- 让打印机检测标签间隙
- 走纸到下一个标签的起始位置
- 确保打印从标签顶部开始

### 为什么第二张正常
第一张打印后，打印机已经完成了标签定位和状态同步，所以第二张通常不会出问题。

### 为什么需要延迟
TSC 打印机的命令是异步的，`printlabel()` 只是发送命令，不会等待打印完成。如果立即发送下一个打印任务，可能会导致缓冲区冲突。
