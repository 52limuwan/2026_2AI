# 打印图片目录

此目录存放下单后自动打印的固定图片。

## 文件说明

- `a1.png` - 第一张打印的图片
- `a2.png` - 第二张打印的图片

## 图片要求

- 格式：PNG、JPG、JPEG、BMP
- 尺寸：建议 400x400 像素（对应50x50mm标签）
- 分辨率：203 DPI

## 使用说明

1. 将你的图片命名为 `a1.png` 和 `a2.png`
2. 放在此目录下
3. 每次下单后会自动打印这两张图片

## 注意事项

- 图片会自动缩放到50x50mm标签大小
- 如果图片不存在，打印会失败
- 支持彩色图片，打印时会自动转为黑白
