# 语音通话思考状态和技能动画功能

## 功能概述

为语音通话界面添加了"思考中"状态指示器和技能动画效果，提升用户体验。

## 新增功能

### 1. 思考中动画
- 当用户停止说话后，显示"思考中..."文字和三个跳动的圆点动画
- 当 AI 开始回复时，思考动画自动消失
- 视觉效果：三个白色圆点依次上下跳动

### 2. 技能显示
- 当 AI 使用特定技能时，显示技能徽章
- 技能徽章包含星形图标和技能名称
- 激活的技能会有金色高光效果和旋转动画
- 技能显示 3 秒后自动淡出

## 状态流转

```
用户说话 → 停止录音 → 思考中 → AI 回复（显示技能）→ 通话中
```

## 技术实现

### 状态管理
- `isThinking`: 控制思考动画的显示
- `activeSkills`: 存储当前激活的技能列表
- `callStatus`: 新增 'thinking' 状态

### 消息处理
通过 WebSocket 消息触发状态变化：
- `stt` 消息：语音识别完成，进入思考状态
- `llm` 消息：AI 开始回复，停止思考，显示技能
- `tts` 消息：语音合成开始/结束

### 动画效果
1. 思考动画：三个圆点依次跳动（0s, 0.2s, 0.4s 延迟）
2. 技能徽章：淡入 + 缩放效果
3. 技能图标：持续旋转动画
4. 激活状态：金色高光 + 阴影效果

## 使用示例

### 服务器端消息格式

```javascript
// LLM 回复消息（包含技能信息）
{
  type: 'llm',
  text: 'AI 回复内容',
  skills: [
    { name: '营养分析' },
    { name: '健康建议' }
  ]
}

// 或简化格式
{
  type: 'llm',
  text: 'AI 回复内容',
  skills: ['营养分析', '健康建议']
}
```

## 样式定制

可以通过修改以下 CSS 变量来定制外观：

```css
/* 思考动画速度 */
animation: thinkingBounce 1.4s ease-in-out infinite;

/* 技能显示时长 */
setTimeout(() => { /* 淡出 */ }, 3000); // 3秒

/* 技能图标旋转速度 */
animation: skillRotate 2s linear infinite;
```

## 注意事项

1. 技能信息需要服务器在 `llm` 消息中提供 `skills` 字段
2. 如果服务器不提供技能信息，只会显示思考动画
3. 思考动画会在以下情况自动停止：
   - 收到 LLM 回复
   - 收到 TTS 开始消息
   - 用户重新开始说话
