## 共同添加的功能代码

在两个端的 `<script setup>` 部分都需要添加以下函数：

```javascript
// 识别用户消息中的技能
const detectSkill = (message) => {
  for (const [skill, keywords] of Object.entries(skillKeywords)) {
    for (const keyword of keywords) {
      if (message.includes(keyword)) {
        return skill
      }
    }
  }
  return ''
}

// 启动技能动画
const startSkillAnimation = () => {
  currentSkillText.value = ''
  showSkillIndicator.value = false
  
  if (skillAnimationTimer) {
    clearTimeout(skillAnimationTimer)
  }
  
  setTimeout(() => {
    currentSkillText.value = `查看技能 ${activeSkill.value}`
    showSkillIndicator.value = true
  }, 3000)
  
  setTimeout(() => {
    currentSkillText.value = `读取技能 ${activeSkill.value}`
  }, 4800)
  
  setTimeout(() => {
    showSkillIndicator.value = false
    setTimeout(() => {
      activeSkill.value = ''
      currentSkillText.value = ''
    }, 600)
  }, 6400)
}

// 停止技能动画
const stopSkillAnimation = () => {
  activeSkill.value = ''
  currentSkillText.value = ''
  showSkillIndicator.value = false
  if (skillAnimationTimer) {
    clearTimeout(skillAnimationTimer)
    skillAnimationTimer = null
  }
}
```

## 修改handleSend函数

在 `handleSend` 函数中添加技能识别：

```javascript
const handleSend = async () => {
  const content = inputText.value.trim()
  if (!content || isLoading.value || isThinking.value) return

  const userMessage = {
    role: 'user',
    content: content,
    timestamp: Date.now()
  }
  messages.value.push(userMessage)
  inputText.value = ''
  scrollToBottom()

  saveMessageToDatabase(userMessage)

  // 识别技能
  const detectedSkill = detectSkill(content)
  if (detectedSkill) {
    activeSkill.value = detectedSkill
    startSkillAnimation()
  }

  isThinking.value = true
  
  // ... 其余代码保持不变
}
```

## 总结

按照以上步骤，分别为Guardian端和Gov端添加skill功能。两个端的主要区别在于：

1. **关键词配置不同**：根据各端的用户角色和使用场景配置不同的技能关键词
2. **技能角色不同**：Guardian端面向家属，Gov端面向政府工作人员
3. **其他代码完全相同**：动画效果、识别逻辑、UI展示都一样

这样三个端（Client、Guardian、Gov）都有了完整的AI技能识别功能！
