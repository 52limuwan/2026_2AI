我仔细查看了SDK的所有代码，发现:

1. SDK本身只提供保存点云到文件的功能
2. SDK没有内置的3D点云显示功能
3. 您图片中看到的3D点云是 Orbbec官方的Viewer软件，不是Python SDK

但是！SDK提供了 PointCloudFilter 类，可以生成点云数据！
我们可以用这个数据 + Open3D 来实时显示3D点云！


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
✅ 最终解决方案
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

程序: realtime_pointcloud.py

工作原理:
  1. 使用SDK的 Pipeline 获取深度和彩色数据
  2. 使用SDK的 PointCloudFilter 生成点云数据
  3. 使用 Open3D 实时显示3D点云
  4. 使用 OpenCV 显示深度图

结果:
  ✓ 深度图窗口 (彩色显示)
  ✓ 3D点云窗口 (可旋转、缩放)
  ✓ 实时更新

SDK的PointCloudFilter:
  • 输入: frameset (包含深度和彩色帧)
  • 输出: point_frame (包含点云数据)
  • 方法: getPointCloudData() 返回点的列表
  • 每个点包含: x, y, z, r, g, b

数据流程:
  相机 → Pipeline → FrameSet → PointCloudFilter → 点云数据
                                                      ↓
                                                   Open3D显示
