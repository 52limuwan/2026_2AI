⚠️  重要说明
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

点云不会自动显示！
您需要在程序运行时按键盘按键来生成点云文件。


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
📋 完整操作步骤
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

第1步：启动程序
  python orbbec_viewer.py
  或双击: 启动程序_新环境.bat

第2步：等待图像显示
  会出现两个窗口:
    • 红外图像窗口 (Infrared Viewer)
    • 深度图像窗口 (Depth Viewer)
  
  等待5-10秒，让相机预热

第3步：生成点云
  在程序窗口激活状态下，按键盘按键:
  
  按 R 键  →  生成彩色点云 (RGB Point Cloud)
  按 D 键  →  生成深度点云 (Depth Point Cloud)

第4步：查看生成的文件
  点云文件会保存在程序目录下:
    RGBPoints_20260205_143025.ply
    DepthPoints_20260205_143025.ply

│  按键   │              功能说明                              │
├─────────┼────────────────────────────────────────────────────┤
│    R    │ 生成彩色点云 (RGB Point Cloud)                    │
│         │ • 包含 XYZ 坐标                                    │
│         │ • 包含 RGB 颜色信息                                │
│         │ • 需要深度和彩色数据都正常                        │

│         │ • 只包含 XYZ 坐标                                  │
│         │ • 不包含颜色信息                                   │
│         │ • 只需要深度数据                                   │
│         │ • 文件名: DepthPoints_时间戳.ply                  │
├─────────┼────────────────────────────────────────────────────┤
│    S    │ 保存当前红外图像                                   │
│         │ • 保存为 PNG 格式                                  │
│         │ • 文件名: IR_Image_时间戳.png                     │
├─────────┼────────────────────────────────────────────────────┤
│ ESC / Q │ 退出程序                                           │
└─────────┴────────────────────────────────────────────────────┘

🔍 如何查看点云文件
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

方法1: 使用 CloudCompare (推荐)
  1. 下载安装 CloudCompare
     网址: https://www.cloudcompare.org/
  
  2. 打开 CloudCompare
  
  3. 拖拽 .ply 文件到窗口中
  
  4. 使用鼠标操作:
     • 左键拖动 - 旋转视角
     • 滚轮     - 缩放
     • 右键拖动 - 平移

方法2: 使用 MeshLab
  1. 下载: https://www.meshlab.net/
  2. File → Import Mesh → 选择 .ply 文件

方法3: 使用 Python 脚本查看
  (需要安装 open3d: pip install open3d)
  
  import open3d as o3d
  pcd = o3d.io.read_point_cloud("RGBPoints_xxx.ply")
  o3d.visualization.draw_geometries([pcd])

❌ 常见问题
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

问题1: 按了R或D键没有反应
解决:
  ✓ 确保图像窗口处于激活状态（点击一下窗口）
  ✓ 不要在控制台窗口按键，要在图像窗口按键
  ✓ 等待5-10秒让相机预热
  ✓ 确保两个窗口都有图像显示

问题2: 提示"点云生成失败"
解决:
  ✓ 确保深度窗口有图像
  ✓ 生成彩色点云需要彩色窗口也有图像
  ✓ 重启程序再试
  ✓ 检查相机连接

问题3: 找不到生成的文件
解决:
  ✓ 查看控制台输出，确认是否真的生成了
  ✓ 在程序目录下搜索 .ply 文件
  ✓ 按修改时间排序查找最新文件

问题4: 点云文件很大
说明:
  ✓ 这是正常的！点云文件通常有几MB到几十MB
  ✓ 包含几十万个点的坐标和颜色信息
  ✓ 如果需要压缩，可以使用二进制PLY格式

💡 使用技巧
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 批量采集
   • 可以多次按 R 或 D 键
   • 每次生成的文件名都不同（带时间戳）
   • 不会覆盖之前的文件

2. 选择合适的点云类型
   • 需要颜色信息 → 按 R (彩色点云)
   • 只需要形状 → 按 D (深度点云，文件更小)

3. 最佳采集时机
   • 等待图像稳定后再按键
   • 避免相机移动时采集
   • 确保光照充足（彩色点云）

4. 文件管理
   • 定期整理生成的点云文件
   • 可以创建子文件夹分类存储
   • 重命名文件以便识别

📊 点云文件信息
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

文件格式: PLY (Polygon File Format)
编码方式: ASCII 文本格式
文件大小: 通常 5-50 MB

彩色点云内容:
  • X, Y, Z 坐标 (3个浮点数)
  • R, G, B 颜色 (3个整数 0-255)
  • 每个点一行数据

深度点云内容:
  • X, Y, Z 坐标 (3个浮点数)
  • 每个点一行数据
