## 📄 核心文件详解

### 1. orbbec_viewer.py
**主程序文件**

- **功能**：实时显示红外和深度图像，生成点云
- **类结构**：
  - `OrbbecViewer` - 主类
    - `initialize()` - 初始化相机
    - `run()` - 主循环
    - `display_ir_frame()` - 显示红外帧
    - `display_depth_frame()` - 显示深度帧
    - `generate_rgb_pointcloud()` - 生成彩色点云
    - `generate_depth_pointcloud()` - 生成深度点云
- **依赖**：numpy, opencv-python, OrbbecSDK
- **运行**：`python orbbec_viewer.py`

### 2. config.py
**配置文件**

可自定义的参数：
- SDK路径配置
- 窗口显示设置
- 图像处理参数
- 点云保存选项
- 性能优化参数
- 调试选项

**修改方法**：
```python
# 修改Python版本
PYTHON_VERSION = "3.9"  # 改为 "3.7" 或 "3.8"

# 修改深度图颜色
DEPTH_COLORMAP = "COLORMAP_HOT"  # 改为其他颜色映射

# 禁用某个传感器
ENABLE_IR = False  # 不启用红外流
```

### 3. test_sdk.py
**SDK测试脚本**

- **功能**：验证SDK配置和相机连接
- **测试项目**：
  1. Python版本检查
  2. 依赖包检查
  3. SDK路径验证
  4. SDK模块导入测试
  5. 相机连接测试
- **运行**：`python test_sdk.py`
- **用途**：在运行主程序前诊断问题

### 4. run_viewer.bat
**Windows批处理启动脚本**

- **功能**：
  - 自动检查Python环境
  - 自动安装缺失的依赖
  - 启动主程序
  - 错误提示
- **使用**：双击运行
- **优点**：适合不熟悉命令行的用户

## 🔧 SDK文件说明

### OrbbecSDK核心文件

#### C库文件
- `OrbbecSDK.dll` - SDK核心动态链接库
- `OrbbecSDK.lib` - 导入库

#### Python绑定模块（.pyd文件）
- `Context.cp39-win_amd64.pyd` - 上下文管理
- `Device.cp39-win_amd64.pyd` - 设备控制
- `Pipeline.cp39-win_amd64.pyd` - 数据流管道
- `Frame.cp39-win_amd64.pyd` - 帧数据处理
- `Filter.cp39-win_amd64.pyd` - 滤波器（包括点云）
- `StreamProfile.cp39-win_amd64.pyd` - 流配置
- `Sensor.cp39-win_amd64.pyd` - 传感器控制
- `ObTypes.cp39-win_amd64.pyd` - 类型定义
- `Error.cp39-win_amd64.pyd` - 错误处理

#### Python接口文件（.py文件）
- `ObTypes.py` - 类型和常量定义
- `Pipeline.py` - Pipeline接口
- `Context.py` - Context接口
- `Filter.py` - 滤波器接口
- `Property.py` - 属性管理
- 等等...

## 📦 输出文件

### 点云文件（.ply）
**格式**：ASCII PLY格式

**彩色点云示例**：
```
ply
format ascii 1.0
element vertex 307200
property float x
property float y
property float z
property uchar red
property uchar green
property uchar blue
end_header
0.123 0.456 0.789 255 128 64
...
```

**深度点云示例**：
```
ply
format ascii 1.0
element vertex 307200
property float x
property float y
property float z
end_header
0.123 0.456 0.789
...
```

### 图像文件（.png）
- 格式：PNG
- 内容：红外图像快照
- 命名：`IR_Image_YYYYMMDD_HHMMSS.png`

## 🔄 数据流程

```
相机硬件
    ↓
USB 3.0连接
    ↓
OrbbecSDK.dll (C库)
    ↓
Python绑定 (.pyd模块)
    ↓
Pipeline (数据流管道)
    ↓
┌─────────┬─────────┬─────────┐
│ 深度流  │ 彩色流  │ 红外流  │
└─────────┴─────────┴─────────┘
    ↓         ↓         ↓
┌─────────┬─────────┬─────────┐
│深度图像 │彩色图像 │红外图像 │
└─────────┴─────────┴─────────┘
    ↓         ↓         ↓
    └────┬────┘         │
         ↓              ↓
    点云滤波器      图像显示
         ↓              ↓
    点云数据        OpenCV窗口
         ↓
    保存为PLY文件
```

## 🛠️ 开发建议

### 添加新功能
1. 在`OrbbecViewer`类中添加新方法
2. 在`run()`方法中添加键盘绑定
3. 更新文档说明

### 性能优化
1. 调整`config.py`中的参数
2. 降低流的分辨率
3. 减少点云密度
4. 使用多线程处理

### 调试技巧
1. 设置`LOG_LEVEL = "DEBUG"`
2. 启用`VERBOSE_ERROR = True`
3. 使用`test_sdk.py`诊断问题
4. 查看SDK日志文件
